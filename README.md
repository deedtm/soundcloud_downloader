# SoundCloud Downloader

Простой скрипт на Python для скачивания музыки с SoundCloud. Использует `yt-dlp` под капотом.

---

## Быстрый старт

### 1. Установи Python

Если Python ещё не стоит — скачай с [python.org](https://python.org) и установи. **Важно:** при установке на Windows поставь галочку «Add Python to PATH».

Проверить, что всё ок:
```bash
python --version
```

Должно вывести что-то вроде `Python 3.12.x`.

### 2. Установи yt-dlp

Это утилита, которая ищет и скачивает треки. Ставится одной командой:

```bash
pip install yt-dlp
```

### 3. Скачай скрипт

**Способ 1 — через кнопку Download ZIP (самый простой):**

1. На странице репозитория нажми зелёную кнопку **<> Code** (справа сверху)
2. Выбери **Download ZIP**
3. Распакуй скачанный архив в любую папку
4. Из распакованной папки возьми файл `scdownloader.py` и положи в отдельную чистую папку, где будешь работать

**Способ 2 — если хочешь только файл:**

1. Открой файл `scdownloader.py` на GitHub
2. Нажми кнопку **Raw** (справа над кодом)
3. Нажми `Ctrl + S` (или `Cmd + S` на Mac) и сохрани файл как `scdownloader.py`
4. Положи его в отдельную папку

Больше никаких файлов не нужно — только `scdownloader.py`.

### 4. Подготовь список треков

В той же папке, где лежит `scdownloader.py`, создай структуру:

```
├── scdownloader.py
└── tracks/
    ├── исполнитель1/
    │   └── tracks.txt
    └── исполнитель2/
        └── tracks.txt
```

- `tracks/` — обязательная папка
- Внутри неё — папки с именами исполнителей
- В каждой папке — файл `tracks.txt` со списком треков (каждый с новой строки)

**Формат названий:** желательно `Исполнитель - Название трека` — так скрипт точнее найдёт то, что нужно. Но можно писать и по-своему.

Есть треки без конкретного исполнителя? Создай папку `tracks/MISC/` и положи туда тоже `tracks.txt`.

### 5. Запусти

Открой терминал (командную строку) в папке со скриптом и выполни:

```bash
python scdownloader.py
```

Скрипт сам создаст папку `downloads/`, всё скачает, и в конце запакует в архив.

---

## Что получится

- Все треки в папке `downloads/` — разложены по папкам исполнителей
- Архив `downloads.zip` рядом со скриптом
- Треки сохраняются в MP3, если получится конвертнуть — иначе в исходном формате
- Между запросами есть пауза (1–2 секунды), чтобы не забанили

## Если что-то пошло не так

| Проблема | Решение |
|----------|---------|
| `python` не найден | Переустанови Python с галочкой «Add to PATH» |
| `yt-dlp` не найден | Запусти `pip install yt-dlp` ещё раз |
| Трек не скачивается | Проверь название — попробуй написать точнее или добавить исполнителя |
| Нет папки `tracks/` | Скрипт сам подскажет, что нужно создать |

---

## Настройки (для тех, кто хочет поковыряться)

В начале `scdownloader.py` можно поменять:

| Переменная | Что делает |
|------------|-----------|
| `RATE_LIMIT_RANGE` | Задержка между запросами (по умолчанию 1–2 секунды). При сильно малых значениях треки могут перестать скачиваться |
| `ARCHIVE_FORMAT` | Формат архива: `zip`, `tar`, `gztar` |
